# carrinho_compras_js
🛒 Carrinho JS
🛍️ Sistema de Carrinho de Compras

# 💻 Sobre o projeto

O Carrinho JS é uma aplicação web voltada para a simulação de um carrinho de compras interativo, criada seguindo boas práticas de desenvolvimento Front-end e organização de código.

O sistema possibilita adicionar, excluir e visualizar produtos de forma dinâmica.

Este projeto foi elaborado como parte do processo de aprendizagem no curso Técnico em Desenvolvimento de Sistemas.

# ⚙️ **Funcionalidades e Conceitos Aplicados**

Durante o desenvolvimento, foram utilizados os seguintes conceitos:

**Manipulação do DOM com JavaScript:**
Atualização dos itens do carrinho de forma dinâmica e em tempo real.

**Eventos em JavaScript:**
Utilização de eventos, como *click*, para adicionar e remover produtos.

**HTML Semântico:**
Estruturação do layout seguindo boas práticas de organização.

**CSS Moderno:**
Uso de Flexbox para organização do layout
Aplicação de responsividade com abordagem *Mobile First*
Estilização de componentes

**Lógica de Programação:**
Controle da quantidade de produtos
Cálculo do valor total da compra
Uso de estruturas condicionais

**Versionamento de Código:**
Utilização do Git para gerenciamento e controle de versões
